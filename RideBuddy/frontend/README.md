# RideBuddy Frontend

Angular 18 SPA for the RideBuddy ridesharing platform.

## Stack

- **Framework**: Angular 18
- **Styling**: SCSS with CSS Variables
- **Icons**: Font Awesome 6.5.1
- **HTTP**: Angular HttpClient
- **Routing**: Angular Router
- **Forms**: Reactive Forms

## Features

- User authentication (login/register)
- Search and browse rides
- Create rides (drivers)
- Book rides (passengers)
- Manage bookings and reservations
- Driver ride and booking management

## Development

```bash
npm install
npm start
```

Local dev server: `http://localhost:4200`

## Docker

In the docker-compose setup: `http://localhost:8080`

## Build

```bash
npm run build
```

Output: `dist/ridebuddy-spa`
